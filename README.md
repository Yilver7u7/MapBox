No usar directamente en AngularCLI (a menos que est√©n creadas las variables de entorno), ya que las variables de entorno se crean basados en el .env

## Pasos:
1. Clonar el .env.template y renombrarlo a .env  
2. Llenar las variables de entorno acorde  
3. Crear Angular Envs (opcional)  

```bash
npm run envs
```

4. Para development ejecutar:
```bash
npm run start
```
5. Para produccion ejecutar:
```bash
npm run build
```
